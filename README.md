# 企業流程健診工具 · Process Diagnosis AI

> 輸入部門流程描述，AI 自動診斷痛點、評估自動化可行性，並給出優先改善建議。

🔗 **Live Demo:** https://ethanlinbiz.github.io/process-analyzer/

---

## 這個工具解決什麼問題

企業在評估「哪些流程可以導入 AI 自動化」時，常見的困境是：

- 不知道從哪裡開始分析
- 無法快速識別真正的瓶頸
- 技術評估與業務需求脫節

這個工具模擬顧問訪談的思維流程，讓任何部門主管或業務人員，不需要技術背景，就能在 30 秒內得到一份結構化的流程診斷報告。

---

## 功能說明

| 輸出項目 | 說明 |
|---------|------|
| **痛點診斷** | 識別 3–5 個核心流程瓶頸，說明問題本質與影響 |
| **AI 自動化可行性評估** | HIGH / MID / LOW 評分 + 具體技術可行性說明 |
| **優先改善行動建議** | 從 Quick Win 開始的具體行動步驟 |

---

## 使用方式

1. 前往 [Live Demo](https://ethanlinbiz.github.io/process-analyzer/)
2. 輸入你的 [Anthropic API Key](https://console.anthropic.com/)（僅在瀏覽器本地使用，不會傳送至任何伺服器）
3. 填寫部門資訊、流程描述、遇到的問題
4. 點擊「開始 AI 流程健診」

---

## 技術說明

- 純前端 HTML/CSS/JS，無後端、無資料庫
